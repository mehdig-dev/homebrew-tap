# homebrew-tap

Homebrew formulae for [Shabka](https://github.com/mehdig-dev/shabka) — persistent memory for LLM coding agents.

## Install

```bash
brew install mehdig-dev/tap/shabka
```

## Update

```bash
brew upgrade shabka
```

## What's included

- `shabka` — CLI for searching, managing, and browsing LLM memories
- `shabka-mcp` — MCP server (15 tools) for AI coding assistants
